# 🌟 My Portfolio Website

Welcome to my **Portfolio Website** – a personal project created to showcase my work, skills, and experiences in a beautifully crafted and modern interface.

---

## 🛠 Technologies Used

This project is built using:

- ⚡️ **Vite** – Fast build tool and development server  
- 🧑‍💻 **TypeScript** – Strongly typed JavaScript for better development  
- ⚛️ **React** – Component-based UI library  
- 🎨 **Tailwind CSS** – Utility-first CSS framework for rapid UI development  
- 🧩 **shadcn/ui** – Beautifully designed UI components  

---

## ✨ Features

- Fully responsive modern design  
- Smooth animations and transitions  
- Clean and professional layout  
- Modular and reusable components  
- Fast performance with Vite  
- Easy-to-update content and structure
